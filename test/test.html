This whole background is 3 stacked CSS layers: a dark base color, a repeating grid-line gradient, and a few blurred radial-gradient glow blobs. No images used anywhere.

background-color: #07070c; /* grid */ background-image: linear-gradient(rgba(255,255,255,.06) 1px, transparent 1px), linear-gradient(90deg, rgba(255,255,255,.06) 1px, transparent 1px); background-size: 40px 40px; /* glow (separate layer, blurred) */ background-image: radial-gradient(circle at 15% 25%, rgba(124,58,237,.45), transparent 40%), radial-gradient(circle at 85% 15%, rgba(59,130,246,.35), transparent 40%), radial-gradient(circle at 50% 85%, rgba(20,184,166,.35), transparent 45%); filter: blur(40px);
